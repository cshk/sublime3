# sublime3离线包 + 许可

## 离线包安装：

下载7z包，解压缩
将Package Control B 放到 sublime3--->Browser Packages
在sublime的Package Settings ---> Package Control ---> User 加入
```
	"channels":
	[
		"channel_v3.json的路径"
	],
```

## 许可：

- 修改sublime的数据
- 打开：
```
https://hexed.it/
```
- 将sublime.exe程序上传
- search for  97 94 0D 将这3个值修改为00 00 00
- 导出，覆盖本地文件
